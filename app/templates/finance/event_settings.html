📱 Инструкция iOS Shortcuts

  1. Откройте Shortcuts
  2. Создайте новый Shortcut (+)
  3. Добавьте "URL" → вставьте Webhook URL
  4. Добавьте "Get Contents of URL":
    • URL: переменная URL из шага 3
    • Method: POST
  5. Добавьте "URL" → {{ request.url.scheme }}://{{ request.url.netloc }}/finance/add?event={{ event.id }}
  6. Добавьте "Open URL" → переменная URL из шага 5
  7. Сохраните Shortcut
  8. В Automation → добавьте триггер на уведомление от банка
